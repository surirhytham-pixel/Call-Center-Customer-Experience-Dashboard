
# 📞 Call Center Customer Experience Dashboard

## Overview
An interactive Excel dashboard built to analyze **32,941 customer interactions** across multiple call centers and communication channels. The dashboard provides insights into customer satisfaction, SLA performance, sentiment, and complaint trends.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Visualization

## Key Metrics
- Total Customers: 32,941
- Average CSAT Score: 5.55
- SLA Compliance: 62%
- 4 Call Centers
- 4 Communication Channels

## Key Insights
- Billing Questions were the most common complaint.
- Los Angeles handled the highest call volume.
- Most interactions were resolved within SLA.
- Very Positive customers reported the highest CSAT scores.
